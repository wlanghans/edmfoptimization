# edmfoptimization
Provides data from JAMES publication on EDMF optimization

*of_scm.nc and of_qpm.nc contain the objective function values for the grid-sampled SCM and the grid-sampled quadratic polynomial model, respectively.

*les.nc contains LES benchmark profiles. It holds three profiles for each parameter: 1) BOMEX, average over third hour; 2) CPBL, 20-min average centered around hour three; 3) CPBL, 20-min average centered around hour five
